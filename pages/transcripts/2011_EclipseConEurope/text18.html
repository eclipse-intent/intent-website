At this point, I think that many of you are thinking “If Intent just works for model synchronization, it is quite useless for me, as I don't use models in my software”.

But everything can be represented as a model : a java file, the structure of an Eclipse plugin and its dependencies, the content of a git repository...

You can plug custom synchronizers inside Intent to synchronize whatever you want. I invite you to consult the wiki and check the “Retro” example, that allows to synchronize java test projects.
