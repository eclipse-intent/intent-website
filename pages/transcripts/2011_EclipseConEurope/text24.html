Notes:

First of all we're planning to improve intent core features.

We will first develop a doc export mechanism. The exported doc should be in PDF or HTML, and will provide useful informations like indexes to quickly get the doc parts related to any concept, links between sections...

We'll also improve the editor by adding a complete code completion (inside model fragments) and a preview view (a bit like wikitext), where both textual explanations and model fragments will be rendered in a user-friendly form.

We would also like to populate the Problem view with synchronization and validation issues.

We'll also provide an integration with DOT/Zest, to integrate diagrams inside Intent.

We'll work hard on extensibility, so you can provide your own validation or synchronization quick-fixes (in the demo...).

You will also be able to provide your language/editor to define model fragments (you may use a Xtext editor, or a treeviewer).
