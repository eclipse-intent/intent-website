Notes:

We are eager to integrate Mylyn Tasks and Intent :

=> a first integration will be to embed Intent Document inside Mylyn contexts. The major advantage compared to textile files is that you will only embed the documentations parts that are actually related to your tasks, and not the whole file.

This integration seems quite easy as in fact, an Intent Document is stored as a model, and not a textual file.

=> We would also like to integrate Mylyn Tasks inside Intent documents : that would allow to apply validation rules on any tasks : for example, a bugzilla issue should not be closed if the test has not been written.
